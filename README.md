# Portfólio | Lucas Felix

Portfólio profissional para apresentar projetos full stack, integrações, experiência e canais de contato. O site foi construído como uma página estática rápida, responsiva e fácil de manter no GitHub Pages.

## Online

- Site: [ffelixlucas.github.io](https://ffelixlucas.github.io/)
- LinkedIn: [linkedin.com/in/lucasfanhafelix](https://www.linkedin.com/in/lucasfanhafelix/)

## Objetivo

Centralizar a apresentação profissional em uma página objetiva, com foco em sistemas web, dashboards, pagamentos, automações e projetos aplicados a operação real.

## Destaques

- Seção de projetos com imagens clicáveis e links de demonstração
- Apresentação de stack, integrações e experiência profissional
- Layout responsivo para desktop e mobile
- Interações leves em JavaScript, sem dependências pesadas
- Estrutura otimizada para GitHub Pages

## Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome
- GitHub Pages

## Estrutura

```text
.
├── assets/
│   └── img/        # Foto, previews e imagens do portfólio
├── index.html      # Estrutura principal
├── style.css       # Estilos e responsividade
├── script.js       # Interações da página
└── README.md
```

## Como rodar localmente

Por ser um site estático, basta abrir o `index.html` no navegador ou usar uma extensão como Live Server.

```bash
git clone https://github.com/ffelixlucas/ffelixlucas.github.io.git
cd ffelixlucas.github.io
```
